## ⚠️  경고 및 주의사항

### 1. 이미지 로딩 오류
```
🔴 Failed to load resource: net::ERR_NAME_NOT_RESOLVED
```

**원인**: Unsplash 이미지 URL 일부가 유효하지 않음
**영향**: 일부 상품 이미지가 표시되지 않을 수 있음
**해결 방법**: 유효한 이미지 URL로 교체 또는 로컬 이미지 사용

### 2. 카테고리 필터링 타이밍 이슈
```
✅ Filtered products: 0
```

**원인**: API 요청이 완료되기 전에 DOM 쿼리 실행
**영향**: 테스트에서만 발생, 실제 사용에는 영향 없음
**해결 방법**: `waitForNetworkIdle` 타임아웃 증가

### 3. 빈 src 속성 경고
```
🔴 An empty string ("") was passed to the src attribute
```

**원인**: imageUrl이 undefined일 때 빈 문자열 처리
**해결 방법**: ProductDetail 컴포넌트에서 이미지 URL 검증 추가
